计算机中带符号的整数都是使用二进制的补码

二进制关于10000000如何求出值为-128的10进制;
首先符号位为1代表着是负数, 最终为-128,只是因为-128与普通值不一样,普通负数值是以补码形式存在的,对补码进行减一然后求反得到该值正数的原码,求出十进制值,然后根据符号位决定是负数还是正数;但是-128不能用该方法求值;应该用110000000来计算(-128的二进制是个特殊值)
